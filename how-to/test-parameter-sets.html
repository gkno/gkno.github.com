Test parameter sets
The vast majority of the pipelines have a parameter set called 'test'. This parameter set uses files contained in the tutorial resources bundle that is automatically downloaded when gkno is built. The test parameter set is provided so that the pipeline can be tested without requiring any additional files or knowledge. The command:
  • gkno <pipeline> -ps test
will run the defined pipeline using parameters and files defined in the test parameter and should execute and quickly run to completion. The files associated with the test parameter sets are always small to minimize execution time.
