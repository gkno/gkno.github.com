How to get gkno
Downloading and installing gkno is a very simple process, requiring only a few commands. To begin with, get the code using the following command:
  • git clone https://github.com/gkno/gkno_launcher.git
Running this command will download all of the gkno code and link in all of the tools that are part of gkno, but not actually download or compile the individual tools. To do this and make gkno operational, type the following command (this step will take a little while!):
  • cd gkno_launcher
  • ./gkno build
gkno should now be completely operational. If there were any problems, please see the troubleshooting section at the end of these instructions.
Test gkno
To test gkno, any of the pipelines can be executed using the test parameter set. For example, type:
  • ./gkno freebayes -ps test
