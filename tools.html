Tools
These tools come prepackaged with gkno and are installed during the gkno installation. gkno includes both tools built by us at the Marth Lab and 3rd party tools. These tools can be used to build up a pipeline or simply be launched directly from gkno. *Because of command line conflicts some 3rd party tools can not be directly launched, however all tools can be used in pipelines

Custom Tools
Custom tools can be added to gkno to allow more complex and custom pipelines. We are currently working on tutorials and documentation to make this easier. If you have any issues, please contact us.
