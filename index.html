A bioinformatics pipeline launcher
gkno opens the world of complex bioinformatic analysis to people of all level of computational expertise. This site contains documentation, tutorials and information on all the tools that comprise gkno.
Get gkno
Learn how to download, install and maintain gkno and get data resources.
Example: Call and visualize variants
Use the gkno freebayes pipeline to call and filter variants. Easily parallelize calling over multiple regions and generate a compressed and indexed vcf file. The iobio project allows visualization and real-time analysis of large data sets. Having generated variant calls for your sample, use vcf.iobio to view global summary statistics.
