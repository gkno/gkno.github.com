Glossary
Tools
  • bamtools - alignment manipulation and analysis software.
  • bamleftalign - Tool for left aligning indels in primary alignments.
  • freebayes - Bayesian variant calling software.
  • gatk-lite - Third-party genome analysis software
  • mosaik - Hash-based Smith-Waterman alignment software.
  • ogap - Gap opening realigner that operates on streaming bam files.
  • picard - Third-party genome analysis software
  • premo - Application for determining suitable alignment parameters for Mosaik.
  • samtools - Third party software for sam/bam file manipulation/analysis.
File formats
  • ann - File generated by a neural net and used by the Mosaik alignment software for calculating mapping qualities.
  • bam - Binary version of sam format.
  • bai - Index allowing jumping to specified locations in the bam file.
  • dat - Reference genome in a Mosaik native format. This is a file generated from the fasta file and used within Mosaik for aligning reads against.
  • fasta (fa) - File format for reference genomes. See spec for details.
  • fai - Reference genome index
  • fastq (fq) - File format for sequencing reads and associated base qualities. See spec for details.
  • mkb - Mosaik read archive. Mosaik used a preprocessing step to convert fastq files into this format and then uses these files in the alignment process.
  • sam - Sequence alignment/map format. A format for storing the genomic sequence, position, base and mapping qualities and additional information for sequencing reads. See spec for details.
  • vcf - Variant call format. A file format for storing genomic variants by genome coordinate. As well as allele indormation, file can also contain genotype information for any number of samples. See spec for details.
Variant types
  • complex - a complex allele consisting of multiple phased individual variants.
  • snp - single nucleotide polymorphism. A single base allele that differs from the reference allele at that locus.
  • indel - an insertion or deletion polymorphism.
  • mei - mobile element insertion.
  • mnp - multiple nucleotide polymorphism. A sequence of contiguous nucleotides that differs from the reference allele at that locus.
  • sv - structural variation. Any large scale (>100bp) structural change in the genome, including, but not limited to insertions, deletions, inversions and translocations.
General terms
  • aligning - The process of finding the best position for a sequencing read in a reference genome. After the best position has been found, the read is said to be aligned.
  • phasing - Two alleles are phased if they appear on the same chromosome.
  • mapping - See aligning.
